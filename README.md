# NEXUS AI Support Triage

17-node n8n workflow powered by OpenAI API that classifies incoming support tickets by urgency and topic, sends automated responses for common queries, routes urgent tickets via Switch nodes, and triggers instant Gmail notifications to the relevant team.

## Tech
n8n (self-hosted) | OpenAI GPT API | Gmail | Switch Node | Function Node | Error Handling

## How to Use
1. Import the JSON file into n8n
2. Add your OpenAI API key
3. Connect Gmail credentials
4. Activate the workflow
